Что такое Jekyll?

Jekyll - простой генератор статических сайтов с поддержкой блогов. Он берёт каталог шаблонов содержащий необработанные текстовые файлы в различных форматах, пропускает его через конвертер (например Markdown) и рендер Liquid, и на выходе вы получаете полностью готовый к публикации статический веб-сайт, подходящий для размещения на вашем любимом веб-сервере. Jekyll так же является движком используемым сервисом GitHub Pages, что означает что вы вожете использовать Jekyll размещения страницы, блога или веб-сайта на серверах GitHub абсолютно бесплатно.

Простота установки и публикации

Начните с клонирования исходного кода в учетную запись GitHub вашего проекта. Благодаря GitHub Pages, он будет автоматически скомпилирован и опубликован в поддомене вашей учётной записи (или организации) в github.io.

Модульные стили и шаблонизация

Этот шаблон использует bootstrap-sass вместе с Bootwatch themes. Вы можете изменить тему или написать собственную презаписав переменные sass в начальной загрузке используя свои наборы цветов, параметров шрифта и т.д.

Контроль версий Git

Используйте систему контроля версий Git, дляподдержания вашей документации в актуальном состоянии вместе с исходным кодом; публикуйте страницы прислиянии с основной веткой.
